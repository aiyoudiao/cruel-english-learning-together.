# GitHub 学习打卡助手 (Project Introduction)

一个完全基于 GitHub 构建的进阶学习追踪平台，具备草稿持久化和离线适应性。

## 🌟 核心特性 (V4 学习版)

- **草稿持久化**：永远不会丢失你的工作。草稿会自动保存到本地存储。
- **自动保存**：每秒自动保存更改（防抖）。
- **离线模式**：即使离线也能继续撰写草稿和附加资源。
- **弹性编辑**：会话恢复确保你可以从上次离开的地方继续。
- **状态指示**：直观的保存状态反馈。

## 🚀 快速开始

1.  克隆仓库。
2.  `cd web`
3.  `npm install`
4.  `npm run dev`

## 🛠 技术栈

- **前端**: React, Vite, TailwindCSS, TipTap
- **后端**: GitHub API, GitHub Actions
- **持久化**: LocalStorage, GitHub Repo
